# BLACK RUSSIA Survival v8

Обновление v8:

- обычные вопросы теперь идут по 2 минуты;
- мини-игра реакции идёт 30 секунд;
- таймер показывает нормальный формат `02:00`, а не `00:120`;
- после окончания времени следующий раунд запускается автоматически, если открыт экран игрока или экран трансляции;
- при создании новой игры порядок вопросов перемешивается;
- варианты ответов тоже перемешиваются, правильный ответ сохраняется корректно;
- исправлена мобильная верстка: варианты ответов видны сразу под вопросом, карточка раунда стала компактнее;
- исправлено частое срабатывание звука тайм-аута.

## Важно

Для авто-перехода по окончании времени нужно один раз выполнить свежий SQL:

`supabase/schema.sql`

или файл:

`BR_SURVIVAL_RUN_IN_SUPABASE_v8.sql`

После обновления GitHub Pages замени файлы:

```txt
index.html
styles.css
app.js
assets/
```

Новые настройки раундов применяются к новым созданным играм. Старую игру лучше удалить и создать заново.

Панель ведущего:

```txt
/?host=1
```

Код ведущего:

```txt
JohnWick123
```
