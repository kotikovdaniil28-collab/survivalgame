# BLACK RUSSIA Survival v6

Версия с отдельным экраном трансляции и устрашающими звуками.

## Что добавлено

- Отдельный режим для трансляции на второй браузер: `?screen=INVITE_CODE`
- В панели ведущего появилась отдельная ссылка **Экран трансляции**.
- На экране трансляции нет кнопок управления: только вопрос, таймер, варианты, статистика и таблица игроков.
- Встроены реальные WAV-звуки в `assets/sfx/`:
  - ambient-drone.wav — фоновый тревожный гул;
  - start.wav — запуск игры;
  - round.wav — новый раунд;
  - correct.wav — верный ответ;
  - wrong.wav — ошибка;
  - death.wav — выбывание;
  - victory.wav — победа;
  - tick.wav — тики последних секунд;
  - timeout.wav — конец времени;
  - boom.wav — удар/напряжение.
- Добавлена кнопка звука в шапку.
- Для второго браузера добавлена кнопка **Включить звук трансляции** — её нужно нажать один раз перед стримом, иначе браузер может заблокировать звук.
- Добавлено настоящее лого BLACK RUSSIA в шапку и на экран трансляции.

## Как использовать трансляцию

1. Открой панель ведущего:

```txt
/?host=1
```

2. Введи код ведущего:

```txt
JohnWick123
```

3. Создай или открой игру.
4. Скопируй ссылку **Экран трансляции**.
5. Открой её во втором браузере/окне.
6. Нажми на экране трансляции кнопку **Включить звук трансляции**.
7. Именно это окно показывай зрителям в Discord.
8. В первом браузере управляй игрой: запуск, следующий раунд, завершение.
