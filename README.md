# BLACK RUSSIA Survival v7 Stable Broadcast

Исправленная стабильная версия после теста:

- убрано постоянное дёргание интерфейса;
- звуки заменены на более мягкие напряжённые, без хоррора;
- после ответа зелёным/красным подсвечивается только выбранный игроком вариант;
- остальные варианты после ответа становятся нейтральными, а не зелёными;
- экран трансляции стал спокойнее и чище для Discord/OBS;
- сохранены мобильная оптимизация, лого, 30 раундов и режим ведущего.

## Ссылки

Панель ведущего:

```txt
/?host=1
```

Код ведущего:

```txt
JohnWick123
```

Экран трансляции копируется из панели ведущего после открытия игры.

## Обновление на GitHub Pages

Заменить в репозитории:

```txt
index.html
styles.css
app.js
assets/
```

SQL повторно выполнять не нужно, если база уже работает.
