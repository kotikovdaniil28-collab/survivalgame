# BLACK RUSSIA Survival v5

Браузерная игра на выживание для Discord-трибуны: invite-ссылка, игроки без регистрации, панель ведущего по скрытой ссылке, 30 раундов, жизни, очки, спецраунды, звуки и мобильная оптимизация.

## Что нового в v5

- Очень сильная адаптация под телефоны.
- Уменьшены тяжёлые картинки для мобильной загрузки: `hero-mobile.webp`, `lobby-mobile.webp`, `rules-mobile.webp`.
- Улучшены размеры кнопок и ответов под палец.
- Таймер и заголовок раунда на телефоне закреплены сверху карточки.
- Таблица игроков, лобби, правила, главная и панель ведущего теперь нормально складываются в один столбец.
- Убраны лишние элементы навигации на маленьком экране.
- Добавлена поддержка safe-area для iPhone.
- Добавлен `theme-color` и mobile web-app мета-теги.

## Панель ведущего

Открывается только по прямой ссылке:

```txt
/?host=1
```

Код ведущего:

```txt
JohnWick123
```

## Игроки

Игроки заходят по invite-ссылке вида:

```txt
/?game=INVITE_CODE
```

Регистрация не нужна: только ник.

## GitHub Pages

Залей в репозиторий:

```txt
index.html
styles.css
app.js
assets/
```

Папку `supabase/` после настройки базы лучше не заливать в публичный репозиторий, потому что там есть код ведущего.

## Supabase

Если база уже работает после v4/v3, SQL можно не выполнять заново.

Если появятся ошибки вида `Could not find the function public.br_...`, выполни в Supabase SQL Editor файл:

```txt
supabase/schema.sql
```

Новые вопросы и настройки применяются только к новым созданным играм.
